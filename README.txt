TEAKWAVE PHP TEMPLATE

Struktur hasil konversi:
- index.php
- profil.php
- produk.php
- produk-detail.php
- kontak.php
- includes/config.php
- includes/header.php
- includes/footer.php
- includes/floating-actions.php
- includes/scripts.php

Catatan pemakaian:
1. Letakkan folder assets, produk, dan file pendukung lain di root proyek yang sama dengan file PHP ini.
2. Jalankan melalui server PHP, misalnya XAMPP, Laragon, atau PHP built-in server.
3. Tautan internal sudah diubah dari .html menjadi .php.
4. Navbar, footer, floating button, dan script global sudah dipisahkan ke folder includes agar lebih mudah dirawat.
