TEAKWAVE FRONTEND + BACKEND UTAKATIK TERHUBUNG DATABASE

Struktur utama:
- index.php, profil.php, produk.php, produk-detail.php, kontak.php sebagai frontend.
- api/products.php untuk mengambil data produk aktif dari database.
- api/contents.php untuk mengambil konten berdasarkan slug data-content-slug.
- api/banners.php untuk mengambil banner berdasarkan placement.
- utakatik/ sebagai backend admin.
- database/digitaria_dashboard.sql sebagai file import database.

Cara menjalankan di XAMPP atau Laragon:
1. Buat database MySQL dengan nama digitaria_dashboard.
2. Import file database/digitaria_dashboard.sql ke database tersebut.
3. Sesuaikan koneksi database di utakatik/config/database.php jika username, password, atau nama database berbeda.
4. Letakkan seluruh folder proyek di htdocs atau root server lokal.
5. Buka frontend melalui index.php.
6. Buka backend melalui utakatik/login.php.

Bagian yang sudah dihubungkan:
- Katalog produk di produk.php mengambil data dari tabel products, brands, dan categories melalui api/products.php.
- Produk best seller di halaman utama mengambil products.is_best_seller = 1 dari database.
- Detail produk di produk-detail.php mengambil data dari database berdasarkan id atau slug produk.
- Gambar produk mengambil path dari kolom products.image dan tabel product_images.
- Konten statis yang memiliki atribut data-content-slug mengambil isi dari tabel contents melalui api/contents.php.
- Banner halaman mengambil data dari tabel banners melalui api/banners.php.

Catatan teknis:
- Produk yang tampil di frontend hanya produk dengan status active.
- Konten yang tampil di frontend hanya konten dengan status published.
- Banner yang tampil di frontend hanya banner dengan status active dan periode tanggal yang masih berlaku.
- URL produk menggunakan format /produk-nama-produk dan diarahkan oleh .htaccess ke produk-detail.php?slug=nama-produk.
